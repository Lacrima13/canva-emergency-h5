# 空白画布急诊室｜Canva 可画 AI 设计急诊 H5

这是一个可直接部署到 GitHub Pages 的单文件 H5 项目。

## 部署方式

1. 新建 GitHub 仓库，例如 `canva-emergency-h5`。
2. 上传本项目中的 `index.html` 和 `README.md` 到仓库根目录。
3. 进入 `Settings → Pages`。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main`，Folder 选择 `/root`。
6. 保存后等待 1–2 分钟，访问 GitHub Pages 链接。

## 说明

本作品以 Canva 可画为品牌对象，设计了一款名为《空白画布急诊室》的交互式 H5 页面。作品将用户面对设计任务时的无灵感、不会排版、配色混乱和时间紧迫等问题转化为“设计急诊”的视觉叙事场景。

技术特点：
- 纯 HTML / CSS / JavaScript，无需构建；
- 可部署 GitHub Pages；
- 自带粒子背景、滚动动画、玻璃拟态卡片、Before/After 对比；
- 适配手机竖版 H5 浏览。
